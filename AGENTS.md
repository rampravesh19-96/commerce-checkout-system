# AGENTS.md

## Project goal
Build a production-style portfolio project suitable for a 3-year full stack engineer targeting product and startup roles.

## Stack
- Frontend: Next.js, TypeScript, Tailwind CSS
- Backend: Node.js, Express.js
- Database: MySQL
- ORM: Prisma
- API style: REST
- Auth: JWT-based auth
- Forms: React Hook Form + Zod
- State/data fetching: React Query
- Deployment target: Vercel for frontend, Render/Railway for backend

## Engineering rules
- Use TypeScript everywhere possible.
- Keep code modular and interview-friendly.
- Prefer clear folder structure and readable naming.
- Avoid unnecessary abstractions.
- Add loading, empty, error, and retry states for critical flows.
- Validate input on both frontend and backend.
- Write concise README docs as features are added.
- Keep sample data realistic.
- Add seed scripts for local setup.

## Done criteria
- Project runs locally with documented setup steps
- Lint passes
- Build passes
- Core happy-path user flows work end to end
- README includes architecture, features, screenshots, and tradeoffs
